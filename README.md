# Скрипты автоматизации

## Быстрый старт

1. Клонируем данный проект (если он еще не был склонирован) или обновляем содержимое:
   ```
   git clone https://172.16.2.50/Davydov/rtt-autotests-docs.git
   cd rtt-autotests-docs/
   git pull
   ```
2. Создаем виртуальное окружение (если оно еще не было создано):
   ```
   python3 -m venv .venv
   ```
3. Активируем виртуальное окружение:
   ```
   source .venv/bin/activate
   ```
4. Устанавливаем зависимости (если не установлены):
   ```
   pip install -r requirements.txt
   ```
